# Smart Lie Detector 🔍

An interactive lie detection simulation with cyberpunk aesthetics.

## Features
- Real-time text analysis
- Particle background animation
- Security breach alert system
- Responsive design

## How it works
- Enter a statement in the input field
- Click "Start Detection"
- If input contains Chinese characters: Truth confirmed ✅
- If input is non-Chinese: Security breach alert 🚨
